Net Value

Net Value is a Google Chrome extension that scrapes soccer player data from the web using Beautiful Soup and a Flask API. Users can search for European soccer players and view data regarding their estimated trade valuation, transfer history, and other basic information.

Languages / Frameworks: Python, Flask, HTML, JavaScript, jQuery, CSS, Bootstrap

GroupUp

iPhone application that allows students to organize meetups based on location. Users can view, create, and search groups and keep in touch with a fully featured chat.

Contributors: Tony De La Nuez, Justin Guyton

Languages / Frameworks: Swift, MapKit, Firebase, JSQMessagesViewController
