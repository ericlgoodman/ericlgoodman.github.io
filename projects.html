Net Value

Net Value is a Google Chrome extension that scrapes soccer player data from the web using Beautiful Soup and a Flask API. Users can search for European soccer players and view data regarding their estimated trade valuation, transfer history, and other basic information.

Languages / Frameworks: Python, Flask, HTML, JavaScript, jQuery, CSS, Bootstrap

Spotify Node.js server

I love both listening to and sharing my music, so I made a Node.js API that I can use to easily make requests to the Spotify API without having to go through the two-way process of requesting and receiving an access token before retrieving data. The landing page shows my top artists, tracks, and my public playlists.

Languages / Frameworks: Node.js, JavaScript, jQuery, Spotify Web API Node, HTML, CSS
